# eshop

### 开发环境说明
* 开发平台 ： IntelliJ IDEA
* JDK版本 ： jdk1.7及以上
* 服务器 ： Tomcat7及以上
* 数据库 ： MySQL

使用时创建数据库shop,编码为utf8，用户名root，密码123456，然后导入shop.sql文件完成数据表的创建。
